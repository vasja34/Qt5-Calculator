# Qt5 Basic Calculator

Basic Calculator is an exercise for Python Software Development course.

## Installation

There is none :) just download the files from a release and run it.

## Requirements

```shell
pip install --upgrade PyQt5
```

## Run

```shell
$ python main_calc.py
```
